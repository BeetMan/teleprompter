# 6.4 更新日志

6.4 版本带来提词时的全局快捷键，并修复控制栏在部分情况下被折叠的问题。

## 全局快捷键

- 打开第二屏输出（开始提词）时，Space 播放/暂停、R 回到顶部、F 全屏、↑↓ 进度、←-> 速度 会注册为系统级全局快捷键，即使焦点不在提词器窗口也能控制。
- 关闭第二屏输出时自动注销全局快捷键，不影响其他应用的输入。

## 控制栏

- 修复：当存储的控制栏高度小于内容所需时（例如从旧版本升级或曾拖到很矮），控制栏不再被裁切折叠，会自动展开到完整高度。
- 设置行的镜像、显示模式四个按钮改为紧凑图标按钮，腾出更多空间给字号/位置滑块。

## 工程

- Tauri 引入 `tauri-plugin-global-shortcut`，Electron 使用 `globalShortcut`。
- 新增全局快捷键分发与开关的回归测试，测试总数达到 21 个。
