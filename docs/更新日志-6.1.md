# 6.1 更新日志

相比 6.0 版本，6.1 主要围绕项目维护、构建一致性和回归检查做了改进。6.1.6 起的问题修复已并入 6.2.0 发布，详见 [更新日志-6.2](./更新日志-6.2.md)。

- Tauri 版和 Electron 版统一使用 `6.1.0` 产品版本号。
- 新增 `VERSION` 统一版本文件，并完善版本同步与检查脚本。
- Tauri 主窗口和第二屏不再维护两份重复页面，第二屏改为通过 `index.html?mode=output` 进入输出模式。
- Tauri 和 Electron 的前端入口页面保持一致，减少两边功能漂移。

## 自动化检查

- 新增 `npm run check` 项目级检查命令。
- 新增前端一致性检查，确保 Tauri / Electron 入口页面内容一致。
- 新增 Playwright 冒烟测试，覆盖主界面渲染、稿件编辑显示和第二屏输出模式。
- 测试优先使用系统 Chrome / Edge，避免额外下载浏览器导致检查卡住。

## 文档和发布

- 更新 README、打包说明、发布清单和维护说明。
- 发布前检查流程改为统一运行版本检查、前端同步检查和基础 UI 测试。
