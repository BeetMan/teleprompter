# 6.3 更新日志

6.3 版本围绕桌面端加固和录制场景的实用增强。

## 6.3.1

安全加固与代码清理，无新功能。

- Electron 两窗口开启渲染进程沙箱（sandbox），缩小攻击面。
- Electron 外部链接改由系统浏览器打开，并增加窗口导航兜底，不再在应用内新开带 Node 能力的窗口。
- Tauri 配置内容安全策略（CSP），收紧 output 输出窗口的权限到仅事件监听。
- 清理无用代码与冗余样式。
- 第二屏输出开关增加重入保护，避免双击触发两次。

## 6.3.0

### 桌面端加固

- 单实例锁：双开时不再启动第二个实例抢第二屏，改为聚焦已有窗口（Electron 与 Tauri 两端均支持）。
- Tauri 版按 `F` 切换全屏现在走原生窗口全屏，行为与 Electron 版一致且更稳定。
- 第二屏输出窗口默认置顶，录制时可以盖在 PPT、OBS、Zoom 等窗口之上。

### 功能增强

- 计时显示：控制栏第一行新增「已用时长 / 剩余时长」读数，播放时实时更新，便于把握录制节奏。
- 拖拽文件类型过滤：拖入非 txt/md/markdown/docx 的文件不再把乱码灌进稿件，改为弹出类型提示。

### 工程

- 新增覆盖上述场景的 Playwright 回归测试，测试总数达到 20 个。
