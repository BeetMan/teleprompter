# 打包说明

本文档说明如何在 Windows 上构建 Tauri 版和 Electron 版。

## 环境要求

### 通用

- Windows 10 / Windows 11
- Node.js
- npm

### Tauri 版额外需要

- Rust
- Cargo
- Tauri CLI 依赖会随 `npm install` 安装

## Tauri 版打包

目录：

```text
windows-tauri
```

命令：

```powershell
cd windows-tauri
npm install
$env:Path="$env:USERPROFILE\.cargo\bin;$env:Path"
npm run build
```

输出文件：

```text
windows-tauri/src-tauri/target/release/teleprompter_tauri.exe
windows-tauri/src-tauri/target/release/bundle/nsis/提词器_6.0.0_x64-setup.exe
```

版本号位置：

```text
windows-tauri/package.json
windows-tauri/src-tauri/Cargo.toml
windows-tauri/src-tauri/tauri.conf.json
```

打包前请确保三个文件中的版本号一致。

## Electron 版打包

目录：

```text
windows-electron
```

命令：

```powershell
cd windows-electron
npm install
npm run dist:win -- --config.win.signAndEditExecutable=false
```

输出文件：

```text
windows-electron/dist/提词器 4.6.0.exe
windows-electron/dist/提词器 Setup 4.6.0.exe
```

版本号位置：

```text
windows-electron/package.json
windows-electron/index.html
```

`package.json` 使用完整版本号，例如 `4.6.0`。

`index.html` 中的 `app-version` 使用短版本号，例如 `4.6`。

## 打包前检查

- 稿件导入是否正常
- 第二屏输出是否正常
- 水平镜像和垂直镜像是否正常
- 第二屏是否能用 `Esc` 退出
- 播放、暂停、回到顶部、回到尾部是否正常
- `↑ / ↓` 快速滚动是否会同步到第二屏
- `← / →` 调速是否正常
- 黑底白字、白底黑字显示是否正常
