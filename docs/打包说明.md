# 打包说明

本文档说明如何在 Windows 上构建 Tauri 版和 Electron 版。

## 环境要求

### 通用

- Windows 10 / Windows 11
- Node.js
- npm

### Tauri 版额外需要

- Rust
- Cargo
- Tauri CLI 依赖会随 `npm install` 安装

## Tauri 版打包

目录：

```text
windows-tauri
```

命令：

```powershell
cd windows-tauri
npm install
$env:Path="$env:USERPROFILE\.cargo\bin;$env:Path"
npm run build
```

输出文件：

```text
windows-tauri/src-tauri/target/release/teleprompter_tauri.exe
windows-tauri/src-tauri/target/release/bundle/nsis/提词器_6.2.1_x64-setup.exe
```

版本号位置：

```text
windows-tauri/package.json
windows-tauri/src-tauri/Cargo.toml
windows-tauri/src-tauri/tauri.conf.json
```

打包前请确保三个文件中的版本号一致。

## Electron 版打包

目录：

```text
windows-electron
```

命令：

```powershell
cd windows-electron
npm install
npm run dist:win -- --config.win.signAndEditExecutable=false
```

输出文件：

```text
windows-electron/dist/提词器 6.2.1.exe
windows-electron/dist/提词器 Setup 6.2.1.exe
```

版本号位置：

```text
windows-electron/package.json
windows-electron/index.html
```

`package.json` 使用完整版本号，例如 `6.2.1`。

`index.html` 中的 `app-version` 使用短版本号，例如 `6.1`。

## 版本号同步

Tauri 版和 Electron 版使用统一产品版本号。修改版本号时优先运行：

```powershell
.\scripts\sync-version.ps1 -Version 6.2.1
```

发布前可以运行：

```powershell
.\scripts\sync-frontend.ps1
.\scripts\check-version.ps1
.\scripts\check-frontend-parity.ps1
```

或直接运行完整检查：

```powershell
npm install
npm run check
```

脚本会同步：

- `VERSION`
- `windows-tauri/package.json`
- `windows-tauri/package-lock.json`
- `windows-tauri/src-tauri/Cargo.toml`
- `windows-tauri/src-tauri/tauri.conf.json`
- `windows-tauri/web/index.html`
- `windows-electron/package.json`
- `windows-electron/package-lock.json`
- `windows-electron/index.html`

## 打包前检查

- 稿件导入是否正常
- 第二屏输出是否正常
- 水平镜像和垂直镜像是否正常
- 第二屏是否能用 `Esc` 退出
- 播放、暂停、回到顶部、回到尾部是否正常
- `↑ / ↓` 快速滚动是否会同步到第二屏
- `← / →` 调速是否正常
- 黑底白字、白底黑字显示是否正常
