# 6.0 更新日志

相比 5.4 版本，6.0 主要围绕第二屏输出、同步稳定性和提词显示逻辑做了升级。

## 第二屏输出

- 主屏现在更接近真正的第二屏预览，使用固定 `1280 × 720` 虚拟画布，再按窗口缩放显示。
- 主屏和第二屏的文字换行、字号比例、指示线位置更一致。
- 优化第二屏同步逻辑，减少播放过程中的抖动和卡顿。
- 播放中按 `↑ / ↓` 快速滚动时，会立即同步到第二屏。
- 第二屏顶部字数提示放大，只显示“剩余 X 字”。
- 字数提示会跟随水平/垂直镜像一起翻转。

## 镜像与进度

- 默认开启水平镜像，更适合反射镜/提词器场景。
- 水平镜像后，指示线上的进度圆点会同步反向。
- 水平镜像后，右侧浏览进度条方向也会反向。
- 同时开启水平和垂直镜像时，进度条方向会按最终镜像状态自动处理。

## 文本与导入

- 修复 `docx` 导入时换行不准确的问题。
- `docx` 导入不再人为删除空段落，也不再额外插入空行。
- 导入文本会保留原始多行换行和缩进。
- 提词器显示时支持显示原文中的空行。

## 键盘操作

- `↑ / ↓` 改为快速滚动页面。
- `← / →` 改为调节滚动速度。
- `Shift + ↑ / ↓` 可更大幅度滚动。
- `Shift + ← / →` 可更大幅度调节速度。
- 最大滚动速度从 `160` 提高到 `240`。

## 显示控制

- “行高”改为更直观的“行距”控制。
- 行距现在直接控制每一行的总步进，不再使用额外的 `1 + 行间距` 公式。
- 行距最低值会根据当前字号动态调整，避免文字重叠。

## 性能与体验

- 第二屏播放改为本地平滑滚动，减少频繁跨窗口同步造成的抖动。
- 移除了字数锚点同步方案中的重布局计算，改善播放时的卡顿感。
- 主屏预览和第二屏输出的视觉一致性更好。
