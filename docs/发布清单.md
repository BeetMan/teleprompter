# 发布清单

如果要把项目整理后发给别人，可以按下面的方式准备。

## 源码交接

建议包含：

- `README.md`
- `使用指南.md`
- `docs/`
- `windows-tauri/`
- `windows-electron/`
- `.gitignore`

建议排除：

- `node_modules/`
- `windows-electron/dist/`
- `windows-tauri/src-tauri/target/`
- 旧版本压缩包
- 本机安装工具，例如 `rustup-init.exe`
- 临时素材和实验目录

## 只发可执行文件

如果只是给别人试用，建议发送安装包：

```text
windows-tauri/src-tauri/target/release/bundle/nsis/提词器_6.2.1_x64-setup.exe
```

也可以发送免安装可执行文件：

```text
windows-tauri/src-tauri/target/release/teleprompter_tauri.exe
```

Electron 版对应文件：

```text
windows-electron/dist/提词器 6.2.1.exe
windows-electron/dist/提词器 Setup 6.2.1.exe
```

## 发布前自测

1. 打开程序。
2. 导入 `.txt`、`.md`、`.docx` 文件。
3. 检查空行和段落是否保留。
4. 点击 `第二屏输出`。
5. 检查第二屏是否无底部控制栏。
6. 检查水平镜像、垂直镜像。
7. 检查剩余字数提示。
8. 播放稿件，观察滚动是否顺滑。
9. 播放中按 `↑ / ↓`，确认第二屏同步。
10. 按 `Esc` 关闭第二屏。

## 推荐命名

源码包可以命名为：

```text
teleprompter-source-6.2.1.zip
```

Tauri 安装包可以命名为：

```text
提词器_6.2.1_x64-setup.exe
```

Electron 安装包可以命名为：

```text
提词器 Setup 6.2.1.exe
```
