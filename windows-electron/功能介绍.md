# 提词器 Electron 版功能介绍

提词器 Electron 版是一款桌面提词工具，适合录课、直播、短视频拍摄、演讲、会议发言和反射式提词器场景。主窗口负责编辑稿件和控制播放，第二显示器负责显示干净的提词画面。

当前 Electron 版版本为 `4.6.0`。功能体验与 Tauri 6.0 版保持一致，主要区别是运行框架不同：Electron 版兼容性较好，Tauri 版体积更小、启动更快。

## 稿件导入与编辑

- 支持在应用内直接编辑稿件。
- 支持导入 `.txt`、`.md`、`.markdown`、`.docx` 文件。
- 支持把文件拖拽到窗口中直接读取。
- 导入稿件后，稿件面板保持自动收起，不打断当前预览。
- 支持保留原始文档中的连续空行，提词显示时也会保留空行节奏。
- 自动保存当前稿件和显示设置，下次打开可继续使用。

## 播放控制

- 支持开始、暂停、回到顶部、回到尾部。
- 支持鼠标滚轮微调滚动位置。
- 支持右侧浏览进度条，可拖动快速定位稿件位置。
- 支持键盘快捷键：
  - `Space`：播放 / 暂停
  - `R`：回到顶部
  - `↑ / ↓`：快速滚动页面
  - `← / →`：调节滚动速度
  - `Esc`：在第二屏窗口中退出第二屏输出

## 显示调节

- 支持调节滚动速度、字号、行距、字间距。
- 支持调节文字显示宽度和水平位置。
- 支持调节第二屏指示线位置。
- 支持黑底白字、白底黑字两种显示模式。
- 去除了屏幕边缘渐变效果，画面更直接清晰。
- 自动换行和手动换行采用一致的排版逻辑，长句阅读节奏更稳定。

## 镜像显示

- 支持水平镜像，默认开启，适合反射式提词器。
- 支持垂直镜像，适合特殊安装方向。
- 开启第二屏输出时，镜像效果主要作用于第二屏，主屏保持可读的控制预览。
- 水平镜像后，第二屏顶部字数提示、指示线进度点和浏览进度方向会跟随镜像方向调整。

## 第二屏输出

- 支持打开独立的第二屏输出窗口。
- 优先输出到第二显示器；只有一个屏幕时，也可在当前屏幕测试。
- 第二屏只显示提词内容，不显示底部控制栏。
- 第二屏顶部显示更醒目的剩余字数。
- 中间指示线包含进度点，便于判断当前阅读位置。
- 按钮再次点击可以关闭第二屏输出，第二屏窗口按 `Esc` 也可以退出。
- 关闭程序时会同步结束相关窗口和进程。

## 主屏预览

- 主屏采用与第二屏一致的固定比例预览，让主窗口成为真正的输出预览。
- 主屏和第二屏共享滚动进度，减少因为窗口宽度不同造成的排版不同步。
- 主屏保留完整控制区，可用于编辑稿件、调节参数、控制播放和切换显示模式。

## 适用场景

- 视频录制：主屏控制稿件，第二屏靠近摄像机显示提词内容。
- 直播带货或课程直播：主播看第二屏，助手在主屏调节速度和稿件。
- 演讲发言：主屏作为控制台，外接屏作为演讲者提示屏。
- 反射式提词器：第二屏开启水平镜像，主屏保持正常显示。
