# 提词器使用指南

适用版本：Tauri 6.1 / Electron 6.1

## 1. 基本用途

这是一款桌面提词器工具，适合录课、直播、演讲、视频口播和反射镜提词使用。

主窗口可以作为预览和控制台使用，第二屏窗口可以作为正式输出画面使用。

## 2. 导入和编辑稿件

点击底部的 `稿件` 可以展开编辑区，直接输入或修改提词内容。

点击 `读取` 可以导入稿件文件，支持：

- `.txt`
- `.md`
- `.markdown`
- `.docx`

也可以把文件直接拖到窗口中导入。

导入 `docx` 时会尽量保留原文段落和空行。编辑区中的原文会保留换行、空行和缩进。

## 3. 播放控制

底部播放区提供常用操作：

- `开始 / 暂停`：控制提词滚动
- `回到顶部`：回到稿件开头
- `回到尾部`：跳到稿件末尾
- `第二屏输出`：打开或关闭第二屏窗口

右侧预览区域有浏览进度条，可以拖动快速定位稿件进度。

## 4. 第二屏输出

点击 `第二屏输出` 会打开一个独立输出窗口。

第二屏窗口特点：

- 不显示底部控制栏
- 显示正式提词画面
- 支持水平镜像和垂直镜像
- 顶部显示剩余字数
- 指示线上显示进度圆点
- 按 `Esc` 可以退出第二屏

6.1 版本中，主窗口采用固定比例预览画布，主屏看到的换行、字号、指示线位置会尽量和第二屏一致。

## 5. 镜像设置

可以在右侧显示设置中开启：

- `水平镜像`
- `垂直镜像`

默认开启水平镜像，适合反射镜或专业提词器场景。

如果只是普通显示器或投屏使用，可以关闭水平镜像。

开启镜像后：

- 第二屏文字会按镜像方向显示
- 指示线上的进度圆点会按水平镜像反向
- 浏览进度条方向会根据镜像状态自动调整
- 剩余字数提示也会跟随镜像翻转

## 6. 显示模式

支持两种显示模式：

- 黑底白字
- 白底黑字

黑底白字适合大多数录制和暗光环境。

白底黑字适合需要更接近纸稿或明亮场景的使用方式。

## 7. 文字样式调整

可以调整：

- 字号
- 行距
- 字间距
- 文本宽度
- 水平位置
- 指示线位置

`行距` 表示每一行文字之间的总步进距离，单位是像素。行距最低值会根据当前字号自动调整，避免文字互相重叠。

`宽度` 会影响自动换行。宽度越窄，换行越多；宽度越宽，单行容纳的文字越多。

## 8. 快捷键

常用快捷键：

- `Space`：播放 / 暂停
- `R`：回到顶部
- `F`：全屏
- `↑`：向上快速滚动
- `↓`：向下快速滚动
- `Shift + ↑ / ↓`：更大幅度滚动
- `←`：降低滚动速度
- `→`：提高滚动速度
- `Shift + ← / →`：更大幅度调节速度
- `Esc`：第二屏窗口中退出第二屏；播放时停止播放

编辑稿件时，方向键会优先用于文本编辑，不会触发提词器快捷操作。

## 9. 使用建议

如果用于反射镜：

1. 保持默认水平镜像开启。
2. 根据镜子方向决定是否开启垂直镜像。
3. 用第二屏输出正式画面。
4. 主屏作为预览和控制台。

如果用于普通第二显示器：

1. 关闭水平镜像。
2. 根据显示器大小调整字号、宽度和行距。
3. 使用 `↑ / ↓` 临时修正滚动位置。
4. 使用 `← / →` 快速调节滚动速度。

如果主屏和第二屏看起来不同步，优先检查两边是否都使用当前 6.1 版本，并尽量不要手动改变第二屏窗口比例。

## 10. 常见问题

### 导入 Word 后空行不对

请使用 `.docx` 文件，而不是旧版 `.doc` 文件。当前版本会保留 `docx` 中的段落和空段落。

### 第二屏没有显示

确认系统已经连接第二显示器。如果没有第二显示器，第二屏窗口仍可能打开在当前屏幕上。

### 文字太挤或重叠

增大 `行距` 或减小 `字号`。行距会根据字号设置最低值，但极大字号下仍建议适当增加行距。

### 滚动太慢

提高 `速度`，也可以按 `→` 快速增加速度。

### 第二屏文字方向不对

切换 `水平镜像` 或 `垂直镜像`，直到第二屏在实际观看设备中方向正确。
